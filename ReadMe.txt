The form takes input from different societies, cells and clubs about the details of their event like description of the event, event time and event venue. The data entered is stored in database named ‘tsg’ . The connection to the database is made by a file named ‘databse.php’.It ensures that the correct entry is being given by verifying it with a security code which is stored in ‘password.txt’. The details are stored in a database and displayed in the events tab in the form of cards highlighting the details. ‘Events.php’ display the files.
